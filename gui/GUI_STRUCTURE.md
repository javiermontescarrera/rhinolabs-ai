  Operaciones en /skills                                                                                                                                                 
  ┌──────────┬───────────────────────────────────────────────────────────────────────────┐                                                                               
  │  Acción  │                                Descripción                                │                                                                               
  ├──────────┼───────────────────────────────────────────────────────────────────────────┤                                                                               
  │ Listar   │ Ver todos los skills con status (enabled/disabled, custom/builtin)        │                                                                               
  ├──────────┼───────────────────────────────────────────────────────────────────────────┤                                                                               
  │ Ver      │ Ver contenido completo del skill                                          │                                                                               
  ├──────────┼───────────────────────────────────────────────────────────────────────────┤                                                                               
  │ Crear    │ Nuevo skill con id, nombre, descripción, categoría, contenido             │                                                                               
  ├──────────┼───────────────────────────────────────────────────────────────────────────┤                                                                               
  │ Editar   │ Modificar nombre, descripción, contenido                                  │                                                                               
  ├──────────┼───────────────────────────────────────────────────────────────────────────┤                                                                               
  │ Toggle   │ Enable/disable                                                            │                                                                               
  ├──────────┼───────────────────────────────────────────────────────────────────────────┤                                                                               
  │ Eliminar │ Solo skills custom (los builtin no se pueden eliminar, solo deshabilitar) │
